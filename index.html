About Me


I am a motivated fresh graduate seeking a position as Software Engineer with the opportunity to develop my skills while contributing to creative and challenging solutions. I completed my MS in Compuer Science from San Francisco State University. I worked on "Analysis and Prediction of Walking Performance using Large Scale Data" as my applied research project. Implementing state-of-the-art ML techniques I achieved an AUC score of 85%.

I served as a Software Development Intern for six months at Backpack Technologies, Inc., a Y-combinator backed e-commerce company. I was responsible for end-to-end development and mainenance of a number of features for the web application and internal tools at full stack. This role required a strong capability in handling ambiguous and challenging problems with time and resource constraints.

Working on a number of projects throughout my professional and academic career made me adept at a wide variety of technologies ranging from Java, Python, C, MySQL to React, Django, Laravel, Git, AWS and many more. My goal is to apply my knowledge to improve our lives and make the world a better place.

Ceres

Ceres

Ceres is a web application that helps users minimize food waste by informing them of the amount and expiration dates of the food they possess. It provides functionalities such as the ability to share a fridge among multiple users, creating a shopping list, search for recipes based on current items etc. The input method is made easy by reading the item names from a receipt using OCR. I worked as the team-lead and full-stack developer in a team of six. My responsibilities include designing the product features and database schema. I set up and maintained the database and web server on AWS. My contribution was to develope the key backend functionalities and database interactions. Technologies used in this project are Git, Python, Django, MySQL, AWS, NGINX, uWSGI, Bootstrap, HTML, CSS.
